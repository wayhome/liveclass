Show HN: Compile C to Not Gates

作者: tomhee | 评分: 81 | 评论数: 14 | 发布时间: 2025-01-17 19:36
文章摘要

文章介绍了 **c2fj** 项目，一个将 C 代码编译为 RISC-V 再进一步编译为 **Flipjump** 操作的编译器。Flipjump 是一种基于 NOT 操作的极简指令集。项目展示了任何程序都能被编译成一系列 NOT 操作。 主要功能包括： 1. **编译流程**：C 代码首先被编译为 RISC-V 二进制，再通过一系列步骤编译为 Flipjump 指令（.fjm 文件）。 2. **支持多文件项目**：通过 Makefile 支持复杂项目，提供相关常量如 `C2FJ_GCC_OPTIONS`。 3. **自定义链接脚本**：需要定义堆、栈等内存区域。 4. **系统调用处理**：将 C 的系统调用（如 `exit`、`_putc`）转换为 Flipjump 实现。 5. **优化**：RISC-V 到 Flipjump 的转换经过高度空间优化，每个 RISC-V 指令被映射为 30-40 个 Flipjump 操作，确保能处理大型代码库。 使用示例： - 安装：`pip install c2fj` 和 `sudo apt install picolibc-riscv64-unknown-elf`。 - 编译运行：`python3 c2fj.py file.c`。 项目展示了如何将高级语言程序通过多个中间层转换为极简指令集。

Maze Generation: Recursive Division (2011)

文章摘要

《Mazes for Programmers》介绍了一种生成迷宫的新方法——递归分割算法。与之前介绍的“ passage carver”算法不同，该方法作为“wall adder”从空场地开始，通过不断添加墙壁生成迷宫。具体步骤如下： 1. 从空场地开始。 2. 用墙壁将场地分割为两部分（水平或垂直），并留出一个通道。 3. 对每个分割后的区域重复上述步骤，递归进行，直到达到期望的迷宫精细度。 该算法具有分形特性，理论上可以无限细分。文章提供了一个5x5网格的示例，并逐步展示了算法的执行过程。最后，还给出了用Ruby实现该算法的代码示例，包括选择分割方向的`choose_orientation`方法和递归分割的`divide`方法。读者可以通过JavaScript演示自行体验算法效果。
