作者: modinfo | 评分: 29 | 评论数: 3 | 发布时间: 2025-01-21 00:49
文章摘要

Ruff 是一个用 Rust 编写的高速 Python 代码检查工具和格式化工具，比 Flake8 和 Black 等传统工具快 10-100 倍。它支持 Python 3.13，内置超过 800 条规则，并兼容 Flake8、isort 和 Black 等工具的功能。Ruff 具有内置缓存、自动错误修复、以及对 VS Code 等编辑器的原生支持，还能在 0.4 秒内完成大规模代码库的检查。Ruff 已被广泛应用于 Apache Airflow、FastAPI 等开源项目中，并可通过 PyPI、Homebrew、Conda 等多种方式安装。
